## PRINCIPE FONDAMENTAL : LE FORK

Une pétition classique = un document, une cible, une liste de signatures.
**Résultat** : facile à ignorer.

Une pétition forkable = un manifeste de base que chacun adapte à SA cible, SON territoire, SA situation.
**Résultat** : mille interpellations simultanées, impossible à contenir.

## TACTIQUES DE GUÉRILLA NUMÉRIQUE

### TACTIQUE 1 : L'ESSAIM TWITTER

**Principe** : Saturer les mentions d'une cible pendant 1 heure.

**Exécution** :
1. Coordonner 20-50 personnes minimum
2. Choisir un créneau (ex: mardi 10h-11h)
3. Chacun tweete en mentionnant la cible avec #SystemeBloque
4. Variantes du même message pour éviter le spam-flag
5. Retweeter les autres participants

**Cibles suggérées** :
- Lundi : @edegryse
- Mardi : @GLBouchez
- Mercredi : @FWB_Federation
- Jeudi : Député·e d'opposition (pour l'inciter à agir)
- Vendredi : Média (pour attirer l'attention)

### TACTIQUE 2 : LE FLOOD D'EMAILS

**Principe** : Saturer une boîte mail de cabinet avec des demandes légitimes individuelles.

**Exécution** :
1. Chaque participant envoie un email PERSONNALISÉ (pas de copier-coller identique)
2. Chaque email raconte UNE histoire personnelle différente
3. Chaque email pose UNE question précise différente
4. Le cabinet doit répondre individuellement = surcharge

**Exemple de questions à poser** :
- "Combien de demandes de reconnaissance ont été refusées à cause du moratoire ?"
- "Quel est le profil socio-économique des porteurs de projets bloqués ?"
- "Le moratoire a-t-il fait l'objet d'une étude d'impact sur les publics précarisés ?"
- "Quelles exceptions sont prévues pour les projets portés par des personnes en situation de pauvreté ?"

### TACTIQUE 3 : LA PRÉSENCE PHYSIQUE

**Principe** : Être visible là où les décideurs sont.

**Cibles** :
- Conseil communal (avec intervention en question d'actualité)
- Commission Culture du Parlement FWB (assister en tribune publique)
- Événements publics des ministres concernés
- Siège des partis MR et Engagés

**Kit visuel** :
- T-shirt ou pancarte "SYSTÈME BLOQUÉ"
- Panneau "L'émancipation est réservée aux déjà-émancipés"
- QR code vers le site

### TACTIQUE 4 : LE DÉTOURNEMENT

**Principe** : Utiliser les canaux officiels pour diffuser le message.

**Exemples** :
- Poser une question lors d'un Facebook Live d'un ministre
- Commenter sous chaque publication officielle de la FWB
- Répondre aux tweets des ministres avec le paradoxe
- Participer aux consultations publiques en ligne avec le manifeste

### TACTIQUE 5 : LA DOCUMENTATION

**Principe** : Créer une archive qui servira de preuve.

**À documenter** :
- Captures d'écran de toutes les interpellations
- Réponses (ou non-réponses) des cabinets
- Témoignages de porteurs de projets bloqués
- Chronologie des actions

**Outils** :
- Google Drive partagé (ou Cryptpad pour plus de sécurité)
- Thread Twitter épinglé avec les mises à jour
- Page "Journal de bord" sur le site

## RÈGLES D'OR

1. **Toujours légal** — Pas de violence, pas de menace, pas de harcèlement. La force du mouvement est sa légitimité.

2. **Toujours documenté** — Chaque action est archivée. Ça servira pour le recours juridique et l'historique.

3. **Toujours personnel** — Pas de copier-coller. Chaque message raconte UNE histoire. Le système ne peut pas ignorer des individus.

4. **Toujours en mouvement** — Pas de pause. Chaque semaine, une action. Le silence = l'oubli.

5. **Toujours ouvert** — Pas de chef. Pas de porte-parole unique. Chacun fork et adapte. Le mouvement appartient à tous.

## EN CAS DE RÉPONSE DU GOUVERNEMENT

### Si réponse type "on comprend mais le budget"
→ Contre-argument : "Le coût de l'exclusion dépasse le coût de l'émancipation. 13M€ économisés sur l'EP = combien de millions en aides sociales prolongées ?"

### Si réponse type "le moratoire est temporaire"
→ Contre-argument : "Temporaire pour vous, permanent pour nous. En 2028, j'aurai 3 ans de plus de précarité. Le temps n'est pas le même pour tout le monde."

### Si réponse type "les structures existantes font le travail"
→ Contre-argument : "Les structures existantes décident QUI émanciper et COMMENT. Nous voulons être acteurs, pas bénéficiaires."

### Si réponse type "il y a d'autres voies"
→ Contre-argument : "Lesquelles ? Le moratoire bloque TOUTE nouvelle reconnaissance. Il n'y a pas d'alternative dans le système actuel."

### Si silence total
→ Documenter le silence. Publier. "X jours sans réponse du cabinet Degryse." Le silence devient une arme contre eux.
