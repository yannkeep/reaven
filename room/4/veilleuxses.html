Principes

Code de Conduite

Respect du RGPD. Proportionnalité. Sources ouvertes uniquement. Pas de données personnelles sans consentement. On documente les systèmes, pas les individus. On ne devient pas ce qu'on combat.

Protocole Berkeley (HCDH/ONU) comme standard de preuve numérique.

Pyramide

Données → Sagesse

  • Données : faits bruts, chiffres BASE
  • Information : données contextualisées ANALYSE
  • Connaissance : compréhension SYNTHÈSE
  • Sagesse : action éclairée DÉCISION

Transformer les données publiques en connaissances utiles à l'action collective. C'est tout.
