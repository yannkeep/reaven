GUIDE — PERSONNALISER STATION ZERO

Comment ajouter des dossiers, créer de nouvelles catégories, et personnaliser votre dashboard de renseignement.

1. Ajouter un nouveau dossier (module)

Ouvrez le fichier dashboard.html dans un éditeur de texte. Trouvez le tableau modules (vers la ligne 80 du script). Ajoutez un objet à la fin du tableau :

// ═══ AJOUTEZ VOS DOSSIERS CI-DESSOUS ═══
{
  id: "mon-nouveau-dossier",       // identifiant unique (pas d'espaces)
  title: "Titre affiché",             // le nom dans la liste
  file: "mon-fichier.html",            // le fichier HTML à charger
  category: "humint",                  // la catégorie (voir ci-dessous)
  kind: "rapport",                     // un tag libre (rapport, briefing, outil...)
  icon: "📋",                           // emoji affiché à gauche
  description: "Ma description"         // texte gris sous le titre
},
IMPORTANT

N'oubliez pas la virgule après l'accolade fermante }, sauf si c'est le dernier élément du tableau. Placez votre fichier HTML dans le même dossier que dashboard.html.

3. Créer une NOUVELLE catégorie

C'est un processus en 3 étapes. Prenons l'exemple d'une catégorie geoint (renseignement géospatial) en jaune.

1 Ajouter la catégorie dans catConfig

Trouvez l'objet catConfig dans le script et ajoutez votre catégorie :

const catConfig = {
  humint:  { label: "HUMINT",  tagClass: "tag-humint",  iconClass: "humint" },
  osint:   { label: "OSINT",   tagClass: "tag-osint",   iconClass: "osint" },
  // ... les autres catégories ...
  analyse: { label: "ANALYSE", tagClass: "tag-analyse",  iconClass: "analyse" },

  // ▸ NOUVELLE CATÉGORIE
  geoint:  { label: "GEOINT",  tagClass: "tag-geoint",  iconClass: "geoint" },
};

2 Ajouter les styles CSS

Dans la section <style> du dashboard, ajoutez les classes de couleur :

/* Dans la section des .tag-xxx */
.tag-geoint {
  background: rgba(234,179,8,0.1);
  color: #eab308;
  border: 1px solid rgba(234,179,8,0.2);
}

/* Dans la section des .module-icon.xxx */
.module-icon.geoint {
  background: rgba(234,179,8,0.08);
  border-color: rgba(234,179,8,0.2);
}

3 Ajouter le bouton de filtre

Dans le HTML, trouvez la div#filterRow et ajoutez un bouton :

<div class="filter-row" id="filterRow">
  <button class="filter-btn active" data-cat="all">Tous</button>
  <button class="filter-btn" data-cat="humint">HUMINT</button>
  <!-- ... les autres boutons ... -->
  <button class="filter-btn" data-cat="geoint">GEOINT</button>
</div>
RÉSULTAT

Votre nouvelle catégorie GEOINT apparaît maintenant dans les filtres, avec sa couleur jaune, et vous pouvez y assigner des modules avec category: "geoint".

4. Ajouter un embed (panneau droit)

Trouvez le tableau embeds et ajoutez un objet :

const embeds = [
  // ... les embeds existants ...
  { id: "mon-outil", icon: "🛠️", label: "Mon Outil", url: "https://example.com", desc: "Description" },
];

5. Modifier l'apparence globale

Les couleurs principales sont définies dans les variables CSS en haut du fichier :

:root {
  --phosphor: #00ff88;      // couleur principale (vert)
  --amber: #ffb000;         // couleur d'alerte (ambre)
  --red-alert: #ff2a2a;     // couleur de danger
  --cyan: #00d4ff;          // couleur secondaire
  --bg-abyss: #050a0e;      // fond principal
  --bg-deep: #0a1118;       // fond des panneaux
}

Pour changer le thème, modifiez ces variables. Par exemple, pour un thème ambre militaire, remplacez --phosphor: #ffb000.
