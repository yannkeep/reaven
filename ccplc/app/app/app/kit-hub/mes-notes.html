Cette page est un exemple. Elle s'affiche quand tu cliques sur un dossier dans ton hub.

Pour créer tes propres pages, il suffit de créer un fichier .html dans le même dossier que ton hub, puis d'ajouter une entrée dans modules.json.

Regarde le fichier GUIDE.html pour tout comprendre pas à pas.
