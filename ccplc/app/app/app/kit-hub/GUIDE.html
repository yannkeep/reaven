Crée ton Hub Personnel

Un tableau de bord rien qu'à toi, avec tes fichiers, tes liens, tes outils. Aucune connaissance en code requise.

1

C'est quoi ce truc ?

C'est une page web qui te sert de tableau de bord. Tu l'ouvres dans ton navigateur (Chrome, Firefox, Safari…) et tu retrouves tous tes trucs rangés au même endroit :

  • 📂 Tes dossiers — des pages HTML que tu crées, triées par catégorie
  • 🔗 Tes liens favoris — des sites web qui s'ouvrent directement dans le hub
  • ⭐ Tes favoris — marque les dossiers importants, ils sont mémorisés
  • 🕐 Tes récents — les derniers dossiers ouverts apparaissent en haut
💡 Le plus important Tout fonctionne sans internet, sans serveur, sans inscription. Tu double-cliques sur un fichier et ça marche.
2

3

Comment tester (en 30 secondes)

1

Mets tous les fichiers dans un dossier

Crée un dossier quelque part sur ton ordinateur (exemple : mon-hub sur le Bureau) et mets-y tous les fichiers du kit.

2

Double-clique sur index.html

Ça ouvre le hub dans ton navigateur. Tu devrais voir l'interface avec les dossiers d'exemple.

3

Clique sur un dossier

La page exemple s'affiche. Sur mobile (ou écran étroit), ça ouvre une modale plein écran. Sur grand écran, ça s'affiche à côté.

⚠️ Petite subtilité En ouvrant index.html directement (avec file://), le chargement des fichiers JSON peut ne pas marcher selon le navigateur. Ce n'est pas un problème : le hub a une copie de secours intégrée et fonctionnera quand même. Quand tu mettras le hub en ligne (étape 7), les JSON marcheront parfaitement.
🖥️ Essaie maintenant ! Double-clique sur index.html pour voir le résultat.
4

Personnaliser tes dossiers

Ouvre le fichier modules.json avec un éditeur de texte. Tu vas voir quelque chose comme ça :

[
  {
    "id": "exemple-notes",            ← identifiant unique (pas d'espaces, pas d'accents)
    "title": "Mes Notes Personnelles", ← le nom affiché dans le hub
    "file": "mes-notes.html",           ← le fichier HTML qui s'ouvre au clic
    "category": "perso",               ← la catégorie (pour trier)
    "kind": "notes",                   ← le petit badge affiché
    "icon": "📝",                      ← l'emoji affiché
    "description": "Toutes mes notes"   ← description courte
  }
]

Chaque bloc entre { } est un dossier. Pour modifier un dossier, change simplement le texte entre les guillemets " ".

⚠️ Les pièges à éviter • Chaque valeur doit être entre "guillemets" • Chaque ligne se termine par une , virgule SAUF la dernière du bloc • Chaque bloc { } se termine par une , virgule SAUF le dernier de la liste • La liste est entre [ ] crochets En cas de doute, copie-colle un bloc existant et modifie-le.

5

Ajouter un nouveau dossier

1

Crée ta page HTML

Crée un fichier (exemple : mon-truc.html) dans le même dossier que index.html. Tu peux y mettre ce que tu veux. Si tu ne sais pas quoi mettre, copie un des fichiers exemples et modifie le contenu.

2

Ajoute une entrée dans modules.json

Ouvre modules.json, et ajoute un nouveau bloc. Le plus simple : copie un bloc existant, colle-le, et change les valeurs.

3

Rafraîchis le hub

Recharge la page du hub dans ton navigateur (touche F5 ou Ctrl+R). Ton nouveau dossier apparaît.

Exemple concret — tu veux ajouter une page "Ma Liste de Films" :

// Dans modules.json, ajoute ce bloc :

{
  "id": "mes-films",
  "title": "Ma Liste de Films",
  "file": "films.html",
  "category": "loisirs",
  "kind": "liste",
  "icon": "🎬",
  "description": "Les films que j'ai vus et ceux à voir"
}
💡 Astuce : le bouton + Dans le hub, il y a un bouton + en bas à droite. Il t'aide à générer le JSON pour un nouveau dossier — tu remplis les champs, tu copies le résultat, et tu le colles dans modules.json.

Ajouter un lien favori

C'est pareil, mais dans embeds.json. Les liens apparaissent dans l'onglet "Outils" (sur mobile) ou dans le panneau de droite (sur desktop).

{
  "id": "mon-lien",
  "icon": "🌍",
  "label": "Mon Site Préféré",
  "url": "https://www.example.com/",
  "desc": "Description courte"
}
⚠️ L'adresse doit être complète Toujours mettre https:// au début de l'adresse. Exemple : https://www.youtube.com/ et pas juste youtube.com.
6

Changer le nom, les couleurs, les catégories

Changer le nom du hub

Ouvre index.html avec un éditeur de texte. Cherche (Ctrl+F) le mot MON HUB et remplace-le par le nom que tu veux. Il apparaît à 4-5 endroits dans le fichier.

Changer la couleur principale

Dans index.html, cherche la ligne :

--phosphor: #00ff88;

Change #00ff88 (vert) par la couleur de ton choix :

CouleurCode
● Vert (par défaut)#00ff88
● Bleu cyan#00d4ff
● Violet#a855f7
● Rouge#ff6b6b
● Orange#ffb000
● Rose#f472b6

Ajouter une catégorie personnalisée

Dans index.html, cherche catConfig. Tu vas trouver un bloc comme :

const catConfig = {
  perso:   { label: "PERSO",   ... },
  travail: { label: "TRAVAIL", ... },
  loisirs: { label: "LOISIRS", ... },
};

Copie une ligne, change le nom, et adapte les couleurs. Ensuite tu peux utiliser cette nouvelle catégorie dans modules.json.

7

Mettre en ligne (optionnel)

Si tu veux accéder à ton hub depuis ton téléphone ou le partager, le plus simple est GitHub Pages. C'est gratuit.

1

Crée un compte sur github.com

C'est gratuit. Choisis un nom d'utilisateur simple.

2

Crée un nouveau "repository"

Clique sur le bouton + en haut à droite → "New repository". Donne-lui un nom (exemple : mon-hub). Coche "Public". Clique "Create".

3

Envoie tes fichiers

Sur la page du repository, clique "uploading an existing file". Glisse tous tes fichiers (index.html, modules.json, embeds.json, et tes pages). Clique "Commit changes".

4

Active GitHub Pages

Va dans "Settings" → "Pages" (dans le menu de gauche). Sous "Branch", choisis main et clique "Save".

8

Questions fréquentes

Mon dossier n'apparaît pas ?

Vérifie que le JSON est valide (pas de virgule oubliée, guillemets bien fermés). En cas de doute, colle ton modules.json sur jsonlint.com — il te dira s'il y a une erreur.

Quand je clique sur un dossier, c'est vide ?

Le fichier HTML indiqué dans "file" n'existe probablement pas, ou son nom est mal écrit. Le nom est sensible aux majuscules/minuscules : Notes.html ≠ notes.html.

Comment trouver des emojis ?

Sur Windows : touche Windows + . (point). Sur Mac : Ctrl + Cmd + Espace. Tu peux aussi copier-coller depuis emojipedia.org.

Quel logiciel utiliser pour éditer les fichiers ?

Le Bloc-notes (Windows) ou TextEdit (Mac) marchent, mais c'est pas idéal. Le mieux, c'est d'installer Visual Studio Code (gratuit) — il colore le code et détecte les erreurs automatiquement.

Je peux mettre autre chose que des pages HTML ?

Oui ! Dans les liens favoris (embeds.json), tu peux mettre n'importe quelle URL : vidéo YouTube, Google Sheets, site web… Le hub les affiche dans un cadre intégré.

C'est sécurisé ?

Le hub fonctionne 100% dans ton navigateur. Aucune donnée ne quitte ton ordinateur. Les favoris et récents sont stockés dans le navigateur (localStorage), pas en ligne.

Je peux l'utiliser sur téléphone ?

Oui, il est conçu pour. Si tu le mets en ligne (via GitHub Pages), tu peux l'ajouter à l'écran d'accueil de ton téléphone et ça ressemble à une app.
